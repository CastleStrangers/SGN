# منصة الجالية السورية في هولندا | Syrian Community in the Netherlands (SY-NL)

منصة رقمية متكاملة تهدف لخدمة وتيسير شؤون الجالية السورية في هولندا، وربط الأعضاء ببعضهم البعض، ومشاركة الأخبار، الفعاليات، والفرص التطوعية.

An integrated digital platform aimed at serving the Syrian community in the Netherlands, connecting members, sharing news, events, and volunteering opportunities.

---

## 🏗 هيكلية المشروع (Project Structure)

يتكون المشروع من قسمين رئيسيين:
1. **تطبيق الويب (Web App)**: لوحة تحكم متكاملة وموقع تعريفي مبني باستخدام Next.js 16.
2. **تطبيق الموبايل (Mobile App)**: تطبيق تفاعلي للأعضاء مبني باستخدام Expo React Native.

```text
SGN/                          # ← تطبيق الويب (Next.js 16 + i18n + Tailwind)
  src/
    app/                      # المسارات والمنافذ البرمجية (API Routes)
    components/               # عناصر الواجهات (React Components)
    lib/                      # المكتبات والخدمات المشتركة (AI, DB)
    messages/                 # ملفات الترجمة للويب (ar, en, nl)
  mobile/                     # ← تطبيق الموبايل (Expo React Native)
    app/                      # شاشات التطبيق الرئيسية
    constants/                # الإعدادات والثوابت والألوان
    i18n/                     # ملفات الترجمة للموبايل (ar, en, nl)
    lib/                      # محرك الترجمة والاتصال بالخادم
  prisma/                     # قاعدة البيانات (Prisma Schema & Migrations)
```

## ⚡ الميزات الرئيسية (Key Features)

* **الأخبار والفعاليات**: تصفح وتصنيف الأخبار (أخبار الجالية، هولندا، أوروبا، اقتصاد، ثقافيات) مع قسم مخصص للفيديوهات ومعرض الصور.
* **إدارة العضويات**: نموذج انتساب ذكي يتيح للأعضاء التسجيل والحصول على بطاقة عضوية رقمية تدعم التحقق عبر رمز الـ QR (متوفر الآن في الويب والموبايل).
* **المخزن الآمن وتحليل المستندات (Vault)**: ميزة جديدة تتيح للأعضاء رفع وحفظ صور هوياتهم وخطاباتهم الرسمية بشكل آمن، مع تحليل ذكي فوري باللغة العربية يشرح محتوى الخطاب والإجراءات المطلوبة.
* **الربط الذكي للتطوع (AI-Skill Matcher)**: نظام يقوم بتحليل مهارات العضو وخبراته واقتراح أفضل المهام التطوعية التي تناسبه بدقة.
* **دليل الخدمات المهنية (Services Directory)**: ميزة جديدة تتيح للأعضاء (المقاولين، الحرفيين، المترجمين، إلخ) عرض خدماتهم المهنية لبقية أفراد الجالية، مما يسهل الوصول للكفاءات السورية في هولندا.
* **المساعد الذكي المتخصص**: مساعد مدعوم بـ Ollama مع شخصيات (Personas) متعددة: (مساعد قانوني، مرشد اندماج ولغة، مستشار توظيف وسكن اجتماعي، والناطق الإعلامي للجالية). يدعم البحث في قاعدة بيانات الموقع (RAG) ليشمل أخبار الجالية والفعاليات وأعضاء مجلس الإدارة والخدمات المهنية.
* **التكامل مع شبكات التواصل**: سحب المنشورات تلقائياً من صفحة الفيسبوك الرسمية للجالية وتصنيفها ذكياً بالذكاء الاصطناعي.
* **التصويت والاستبيانات**: نظام استبيانات وتصويت لأعضاء المجتمع، مع عرض مهام تطوعية للمساهمة في أنشطة الجالية.

## 🛠 التقنيات المستخدمة (Tech Stack)

* **Next.js 16** (App Router & React 19)
* **React Native & Expo 54** (للموبايل)
* **TailwindCSS** (للتنسيق البصري للويب)
* **Prisma ORM** (مع SQLite للتطوير المحلي، و Turso/libsql للإنتاج)
* **Next-Auth** (إدارة الجلسات والصلاحيات)
* **Ollama** (الذكاء الاصطناعي المحلي باستخدام نموذج `qwen2.5:7b` للنصوص و `llava:7b` لاستخراج بيانات الهويات)
* **Next-Intl & useI18n** (دعم كامل للغات: العربية، الهولندية، والإنجليزية مع ملاءمة واجهات RTL/LTR)

## 🚀 تشغيل المشروع محلياً (Local Development)

### 1. إعداد تطبيق الويب (Next.js)

1. قم بتثبيت الاعتماديات:
   ```bash
   npm install
   ```
2. تهيئة ملف البيئة `.env` وإضافة قاعدة البيانات المحلية:
   ```env
   DATABASE_URL="file:./prisma/dev.db"
   ```
3. تطبيق ومزامنة جداول قاعدة البيانات:
   ```bash
   npm run db:push
   ```
4. تشغيل خادم التطوير المحلي:
   ```bash
   npm run dev
   ```
   * سيعمل الموقع على الرابط: `http://localhost:3000`

### 2. إعداد تطبيق الموبايل (Expo)

1. انتقل لمجلد الموبايل وقم بتثبيت الحزم:
   ```bash
   cd mobile
   npm install
   ```
2. لتشغيل التطبيق في بيئة التطوير المحلية (Local):
   ```bash
   npm run set:dev
   npm start
   ```
3. لبناء التطبيق للإنتاج (Production):
   ```bash
   npm run set:prod
   ```
   * سيقوم هذا الأمر بتحويل رابط الخادم تلقائياً ليعتمد على رابط الإنتاج `https://sy-nl.org`.
