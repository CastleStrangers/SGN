🎯 QUICK REFERENCE
═══════════════════════════════════════════════════════════════

Data Operations:
  npm run sync              → Sync Facebook posts
  npm run import:facebook   → Import historical posts
  npm run reclassify        → Reclassify all posts
  npm run db:reset-sync     → Reset and resync

Management:
  npx tsx scripts/05-manage-users.ts list           → List users
  npx tsx scripts/05-manage-users.ts promote <email> → Make admin
  npx tsx scripts/05-manage-users.ts demote <email>  → Remove admin
  npx tsx scripts/05-manage-users.ts delete <email>  → Delete user

Monitoring:
  npm run check-db         → Database health
  npm run diagnose-fb      → Facebook API test
  npm run i18n:check       → i18n validation
